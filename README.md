# 🎯 Project Objective

The goal of this project was to understand:

- Which IPL franchises overpay or buy efficiently
- How player performance influences auction prices
- Which players were undervalued or overpriced
- How auction spending trends changed across seasons

---

# 📂 Dataset Information

This project uses 10 years of IPL data (2013–2022):

### 📌 Raw Datasets
- deliveries.csv
- matches.csv
- IPLPlayerAuctionData.csv

### 📌 Processed Datasets
- player_stats.csv
- final_auction_analysis.csv

# 🚀 Key Features

- Cleaned and transformed 3 messy IPL datasets using Excel and Python
- Generated batting and bowling statistics from raw delivery-level data
- Created a correlation-based weighted scoring model for player valuation
- Compared fair player value against actual auction price
- Identified overpaid and undervalued players across franchises
- Built an interactive dashboard with KPI cards, filters, and analytical visuals

---

# 📊 Dashboard Insights

- Negative average difference suggests franchises generally purchased players below estimated fair value
- Royal Challengers Bangalore recorded the highest overall auction spending
- Strike rate has weak influenceon auction pricing compared tooverall runs and match participation.
- Franchises distributed auction spendingalmost equally between indian and overseas players.
